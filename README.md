# Fake-News-Detector
🇮🇳 AI-powered Fake News Detector for Indian WhatsApp users | NLP + Machine Learning | Python, Scikit-learn, Flask | Combating misinformation in India
## 🚀 How to Run
pip install -r requirements.txt
python app.py

## 📊 Model Performance
- Accuracy: 94.76%
- Dataset: IFND Indian Fake News Dataset
- Total news trained: 56,714
- Real news: 37,800
- Fake news: 18,914

## 🛠️ Tech Stack
- Python
- Pandas + NumPy
- Scikit-learn (NLP + ML)
- Flask
- HTML + CSS
